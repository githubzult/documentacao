    Instalação do MkDocs

    O MkDocs é uma ferramenta de criação de documentação em Markdown que gera um site estático a partir de um conjunto de arquivos Markdown. Neste guia, vamos mostrar como instalar o MkDocs em um ambiente Unix, como Linux ou macOS.

    Pré-requisitos

    Antes de instalar o MkDocs, você precisa ter instalado o Python e o pip.

    Instalando o MkDocs

    Agora que você tem o Python e o pip instalados, pode instalar o MkDocs com o pip. Para instalar o MkDocs, siga os seguintes passos:

    1. Abra o terminal.
    2. Digite o seguinte comando para instalar o MkDocs:

    sudo pip3 install mkdocs

    Este comando instalará o MkDocs e todas as suas dependências.

    1. Para verificar se o MkDocs foi instalado corretamente, digite o seguinte comando:

    mkdocs --version

    Este comando deve imprimir a versão atual do MkDocs.

    Criando um novo projeto MkDocs

    Depois de instalar o MkDocs, você pode criar um novo projeto. Para criar um novo projeto MkDocs, siga os seguintes passos:

    1. Abra o terminal.
    2. Navegue até a pasta onde deseja criar o novo projeto.
    3. Digite o seguinte comando para criar um novo projeto:

    mkdocs new my-project

    Este comando criará uma nova pasta chamada my-project, contendo um arquivo de configuração mkdocs.yml e uma pasta docs, onde você pode adicionar seus arquivos Markdown.

    Iniciando o servidor de desenvolvimento MkDocs

    Depois de criar um novo projeto MkDocs, você pode iniciar o servidor de desenvolvimento para visualizar o site. Para iniciar o servidor de desenvolvimento, siga os seguintes passos:

    1. Abra o terminal.
    2. Navegue até a pasta do seu projeto.
    3. Digite o seguinte comando para iniciar o servidor de desenvolvimento:

    mkdocs serve

    Este comando iniciará o servidor de desenvolvimento e imprimirá o endereço do servidor no terminal.

    1. Abra o navegador e digite o endereço do servidor na barra de endereço. O site deve ser carregado no navegador.

    Gerando o site estático

    Depois de adicionar e editar seus arquivos Markdown, você pode gerar o site estático final. Para gerar o site estático, siga os seguintes passos:

    1. Abra o terminal.
    2. Navegue até a pasta do seu projeto.
    3. Digite o seguinte comando para gerar o site estático:

    mkdocs build

    Este comando gerará o site estático.
